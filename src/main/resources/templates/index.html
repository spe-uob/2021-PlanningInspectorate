Introduction

Hello! Welcome to the Planning Inspectorate Database! The intent of this application is to make it easier to create consultation lists and maintain your contacts' details. You can chose what you want to do using the navigation bar on the left. This was created by computer science students at the University of Bristol.

Create New Consultation List

To create your consultation list you start by uploading the output of the GIS software. This will output the list of government bodies in the area. You can then decided whether to consult them using the tests and notes corresponding to that department. If you need to consult with other organisations (such as the Environment Agency) you can add these afterwards.

Manage Database

This page is used to access and edit the database. You can search for a specific record. Using the edit button on each record you can update or delete it. The 'Add Record' button allows you to add a button to the database manually.

View Form Responses

This allows you to keep track of the updates to the database made by the contacts themselves. You can then select the one you wish to see and are able to edit their requests. You can then use the buttons to accept or deny the changes.
